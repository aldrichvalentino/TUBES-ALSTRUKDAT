/* Cara meng-compile games Bright Souls */

> Buka terminal 
> Lakukan akses ke folder yang dibutuhkan dengan menggunakan 'cd (folder)'
> Tuliskan 'gcc -o mprogram mprogram.c jam.c narasi.c fileeksternal.c matriks.c 
battle.c stack.c queue.c enemy.c player.c point.c map.c graph.c gambar.c listlinier.c mesinkata.c mesinkar.c -lm'
> Setelah berhasil di compile, tuliskan './mprogram'
